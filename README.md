# Digital Process Automation Portal

A MERN-based system to automate academic workflows including task creation, submission tracking, and evaluation.

## Tech Stack
React, Node.js, Express, MongoDB

## Installation

### Clone repository
git clone <repo-url>

### Backend setup
cd server
npm install
npm run dev

### Frontend setup
cd client
npm install
npm run dev

## Environment Variables

Backend (.env)
PORT=5000
MONGO_URI=your_mongo_url
JWT_SECRET=secret

Frontend (.env)
VITE_API_URL=http://localhost:5000
